3d cube

This project is to make and export a 3d object to the game engine in the proper way. It is to make a cube with multiple images and the size is 1 x 1 x 1 in the game engine. I used 3d Max to create the object. Tasks I will do include:l assign materials through the material editor, render a scene, render to texture, and export the Model to an image.

3D Kangaroo

This project utilized many skills for mesh modeling. I used 3d Max to create the object. The first step is to make the reference planes. Then, I applied the box approach to create 5x3x1 segments. Body editing was the next step. I edited the arms and legs position. The last part I edited was the head.

3D Character Animation

In this project, I utilized many skills to make the kangaroo move and dance. I used 3d Max functions to achieve that. Functions included Character Animation Tool (CAT) Parent, Skin Modifier, and CAT Motion Layer, Envelop Editing in Skin modifier, Vertex Weight Editing with the Weight Table in Skin Modifier, and CAT Motion Layer Editing. The walking and running animations are created in the CAT motion layer.

Create Walking Animation in MotionBuilder

In this project, I used the MotionBuilder, which is the top line product to make motions in Hollywood movie and game studios in the world, to create walking animation. Tasks I performed include adding auxiliary pivot for Toe, FootBase, and Heel in order to make realistic walking animation, making 4 poses; Heel-Contact, Squash, Passing, and Up, mirroring the 4 poses, and editing the walking motion at one position.

Motion Capture Data in Motion Builder

In this project, I made motions from motion-captured (MoCap) data using the other template, Actor. An actor is a template of Characters item. I imported the MoCap data into MOBU, and associated the point cloud data to the makers of the actor. Once mapping between the makers and the points, the points move the actor's each piece in animation frames. I created a FBX file, which was tested in Unity. All motions were glued into the skeleton.
